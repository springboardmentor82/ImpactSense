# Week 5: Model Evaluation & Explainability

## Objective
The objective of Week 5 is to evaluate the performance of the trained machine learning model and understand how different features influence the predictions. This week focuses on model evaluation metrics and explainability techniques to interpret the model's behavior.

## Tasks Performed
- Evaluated the trained Gradient Boosting model using test data
- Generated a Confusion Matrix to visualize classification performance
- Calculated evaluation metrics including:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
- Created visualization plots for MAE and MSE
- Implemented Feature Importance analysis to identify which input features contribute the most to the model predictions
- Applied SHAP (SHapley Additive exPlanations) to interpret the model and understand feature impact
- Generated a SHAP summary plot to visualize how each feature affects the prediction outcome

## Results
- The Confusion Matrix shows the classification performance of the model across different classes
- MAE and MSE provide quantitative measures of prediction error
- Feature Importance analysis highlights the most influential features affecting model predictions
- SHAP analysis provides a deeper understanding of how each feature contributes positively or negatively to the final prediction

## Outcome
- Model evaluation was successfully performed using multiple metrics
- Model interpretability techniques were applied to understand feature contributions
- Visual explanations were generated for better understanding of the model behavior
- The machine learning pipeline is now complete with both performance evaluation and explainability.
