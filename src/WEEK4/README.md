# Week 4: Advanced Model Training & Optimization

## Objective
The objective of Week 4 is to train advanced ensemble machine learning models using the processed dataset and improve prediction performance through hyperparameter tuning and cross-validation.

## Tasks Performed
- Loaded training and testing datasets generated from previous preprocessing steps
- Implemented **Random Forest Classifier** as an ensemble learning model
- Implemented **Gradient Boosting Classifier** for improved predictive performance
- Tuned Gradient Boosting parameters to optimize model accuracy
- Performed **cross-validation** on Random Forest to evaluate model stability
- Evaluated models using **accuracy score**
- Tested the trained model using a **sample input prediction**

## Results
- Random Forest achieved strong performance by combining multiple decision trees
- Gradient Boosting achieved higher accuracy after parameter tuning
- Cross-validation confirmed that the model performs consistently across different data splits

## Outcome
- Advanced ensemble models were successfully trained and evaluated
- Gradient Boosting achieved the best performance among the tested models
- The optimized model is ready for further **evaluation and explainability analysis in Week 5**
