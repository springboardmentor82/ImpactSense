# Week 1: Project Setup & Dataset Understanding

## Objective
The objective of Week 1 is to understand the earthquake dataset and explore its structure, features, and basic patterns before applying any preprocessing or machine learning models.

## Tasks Performed
- Loaded the earthquake dataset using Pandas
- Explored dataset structure, shape, and column names
- Checked dataset information and data types
- Identified missing values
- Removed duplicate records
- Performed Exploratory Data Analysis (EDA)
- Visualized:
  - Magnitude distribution
  - Depth distribution
  - Alert level distribution
- Generated a correlation heatmap for numerical features

## Outputs
- Console outputs showing dataset structure and missing values
- Visual plots for magnitude, depth, and alert distribution
- Correlation heatmap showing relationships between seismic features

## Outcome
- Gained clear understanding of the dataset
- Identified important seismic features
- Generated a cleaned dataset (`earthquake_cleaned.csv`) for further processing
