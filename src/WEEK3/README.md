# Week 3: Baseline Model Training & Evaluation

## Objective
The objective of Week 3 is to train baseline machine learning models and evaluate their performance using the preprocessed dataset from Week 2.

## Tasks Performed
- Loaded training and testing datasets
- Identified the task as a multi-class classification problem
- Implemented Logistic Regression as a baseline linear model
- Implemented Decision Tree Classifier to capture non-linear relationships
- Trained both models using the training dataset
- Evaluated models using:
  - Accuracy score
  - Confusion matrix
  - Classification report (Precision, Recall, F1-Score)

## Results
- Logistic Regression achieved moderate accuracy and served as a baseline model
- Decision Tree achieved higher accuracy due to its ability to model complex feature interactions
- Decision Tree outperformed Logistic Regression across most evaluation metrics

## Outcome
- Baseline models were successfully trained and evaluated
- Decision Tree identified as the better-performing model
- Project is ready for advanced models and further optimization
