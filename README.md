# \# ImpactSense

Earthquake Prediction System

A machine learning-based web application to predict earthquake impact levels based on parameters like magnitude, depth, CDI, MMI, and significance.

🔗 Live Demo: https://impactsense-app-gtvq.onrender.com

##  Project Structure
- Week 1 – Data Collection
- Week 2 – Data Cleaning
- Week 3 – Model Building
- Week 4 – Evaluation
- Week 5 – Final Model
- Week 6 – UI Prototype
