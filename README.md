👉 https://impactsense-3zlk.onrender.com

## 📌 Project Description
ImpactSense is a machine learning-based web application that predicts earthquake magnitude based on seismic parameters.

---

## ⚙️ Features
- Earthquake magnitude prediction
- REST API using Flask
- Real-time deployment using Render
- Interactive UI (if added)

---

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
- HTML, CSS (Frontend)
- Render (Deployment)

---

## 📂 Project Structure
- app.py → Main Flask app
- model.pkl → Trained ML model
- requirements.txt → Dependencies
- runtime.txt → Python version config
