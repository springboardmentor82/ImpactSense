# 🌍 ImpactSense - Earthquake Prediction System

## 🔗 Live Demo

👉 https://impactsense-3zlk.onrender.com

---

## 📌 Project Description

ImpactSense is a machine learning-based web application that predicts earthquake magnitude based on seismic parameters. It helps in analyzing earthquake impact levels and provides insights using trained models.

---

## 🚀 Features

* 🌎 Earthquake magnitude prediction
* ⚡ REST API using Flask
* 📊 Machine learning model integration
* 🌐 Deployed using Render
* 💻 Simple and interactive UI

---

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas, NumPy
* HTML, CSS
* Render (Deployment)

## 📘 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Harshith32424/ImpactSense.git
```

2. Navigate to folder:

```
cd ImpactSense
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
python app.py
```
