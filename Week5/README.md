# 📊 Week 5: Evaluation & Explainability

## 🎯 Objective

To evaluate the machine learning model and understand its decision-making process using explainability techniques.

---

## 🔍 Features Implemented

### ✅ Model Evaluation

* Confusion Matrix
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)

### 📈 Visualization

* Error comparison graph
* Confusion matrix heatmap

### 🧠 Explainability

* Feature Importance
* SHAP (SHapley Additive exPlanations)

---

## 🖥️ Web Interface

A simple Flask-based dashboard to display:

* Model performance metrics
* Graphical insights
* Explainability outputs

---

## ⚙️ Tech Stack

* Python
* Flask
* Scikit-learn
* Matplotlib & Seaborn
* SHAP

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open browser:

```
http://127.0.0.1:5000/
```

---

## 📌 Output Screens

* Confusion Matrix
* MAE vs MSE Plot
* Feature Importance Graph
* SHAP Summary Plot

---

## 📢 Conclusion

This module evaluates model performance and provides transparency using explainable AI techniques.
